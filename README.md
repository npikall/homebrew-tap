# Homebrew Tap

A repository, to distribute applications via homebrew.

## Installation

First, add this tap:

```bash
brew tap npikall/tap
```

Then install any formula from this tap:

```bash
brew install <formula>
```

Or without adding the tap first:

```bash
brew install npikall/tap/<formula>
```

### Example: gotpm

```bash
brew tap npikall/tap
brew install gotpm
```

Or without adding the tap first:

```bash
brew install npikall/tap/gotpm
```
